There are two types of path that we can access in our hard drive. They are listed below.

  1. Absolute path

    in src="C:\Users\karan\Desktop\web development\structured_way"

  2. Relative path

    3. if image src="./any folder or any file etc after it"
    it means check the required file in the same directory
    where the .html file of the webpage is created say index.htm l


    if image src="../any folder or any file etc after it

    it means to check the required file in the previous directory

    form where the say index . html page is created


    basically ,
    if 1 dot(./) before relative path,then console check within the file where index. html file is stored

    if 2 dot(../) before relative path,then system checks the required file in previous folder from there the say index. html file is created.
